Era · Agent Evaluation

A measurement instrument for agentic chains

Evaluate one Pydantic-AI agent — then a chain of three — quantitatively, with attribution to each component, honest uncertainty, and a deployable routing policy. Every number on this page is produced by the framework at seed 0.

framework-agnostic — Sim · Pydantic-AI · maseval

The one idea

An agent run is response = f(system_prompt, llm, tools | query). Everything between input and output is observable, so it is measurable. Reduce every run — simulated, a real Pydantic-AI agent, or one driven through maseval — to a single RunTrajectory, and do all evaluation on that. The evaluator never needs to know which framework produced the run.

04Reliability — pass^k, not just average quality NEW

Average quality (pass@1) rewards a config that is usually right. pass^k (τ-bench, arXiv:2406.12045) asks: does it solve the task on all k tries? Cheap models look fine at pass@1 and collapse under repetition — the frontier over-selects them unless you measure this.

gpt-4o — 0.96 → 0.76 phi-3.5-mini — 0.78 → 0.29
Same golden set; both look acceptable at k=1. By k=8 the cheap model is 2.6× less reliable. pass^k is now a Pareto axis, so consistency is priced in.

05Failure modes — MAST taxonomy NEW

A score says whether it failed; MAST (arXiv:2503.13657) says how. A deterministic post-processor tags each run into the empirical multi-agent failure taxonomy — and localizes the break in the chain.

06Continuous monitoring — evals → Grafana NEW

The north star is a bench that gates what ships. Beyond the CI pass/fail gate, every eval number is exported as Prometheus/OpenMetrics and pushed to the team's Grafana (telemetry.era.world) — quality, cost, the routing savings, and the pass^k curve, next to the production service dashboards.

07Why you can trust the numbers

Seeded

BLAKE2b digest seeds — process-independent determinism at seed 0.

Bootstrapped

Every metric carries a 95% CI; verdicts require the CI to exclude zero.

Paired

Interventions share seeds, so a Δ is the change — not luck.

Self-validating

Recovers planted ground truth before it's trusted on real models. 71 tests green.
